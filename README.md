# Professional Image Editor

A desktop image editor built using:
- Python
- Tkinter (GUI)
- OpenCV (Image Processing)
- Object-Oriented Programming principles

## Features
- Grayscale conversion
- Gaussian blur (adjustable)
- Edge detection
- Brightness & contrast adjustment
- Rotate, flip, resize
- Undo / Redo
- Save & Save As

## How to Run
```bash
python main.py
